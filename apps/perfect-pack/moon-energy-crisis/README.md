# 月球能源危机

面向 10–15 岁玩家的移动端科学策略游戏。

玩家需要追踪不断移动的最佳光照窗口，同时在氧气、通讯和探测车之间分配有限电力。三回合逐步增加目标速度与能源事件，让太阳能知识成为获胜规则，而不是答题内容。

## 核心循环

1. 点击、拖动或使用方向键追踪最佳光照窗口。
2. 观察实时发电量、总需求和电力缺口。
3. 关闭次要设备，保持氧气系统稳定供电。
4. 连续稳定供电以累积连击和得分。
5. 完成三回合并刷新本地最佳成绩。

## 本地运行

```bash
npm install
npm run dev
```

## 数据

游戏只在当前浏览器 `localStorage` 保存最佳得分，不上传身份或个人信息。
