# 谁最能装？

移动优先的空间拼图小游戏。玩家从传送带挑物品，旋一旋、塞一塞，在倒计时结束前把箱子填满，看看谁才是真正的装箱天才。

## 玩法

1. 点击传送带上的物品。
2. 点箱格选一个开塞位置。
3. 左右旋转，找到那个“刚刚好”的角度。
4. 青色预览亮了就大胆塞进去。
5. 五局越塞越难，发给朋友比比谁更能装。

每一局都由完整布局反向生成，所以一定存在 100% 满格解法。难点会逐步增加：箱子更大、东西更多、形状更刁钻、时间更紧张。

## 分享挑战 MVP

- 每关完成后生成个人战绩卡。
- 支持本地保存昵称、随机短标记和可选宣言。
- 战绩图片为 `1080 × 1440`，展示成绩、二维码和可爱物品堆，不展示最终摆法，避免透题。
- 挑战链接只携带关卡和分享者成绩，打开后直接挑战同一局。
- 挑战完成后比较双方用时和得分。
- 第一版无需账号或后端，挑战数据编码在链接中。
- 微信内不调用文件下载：打开高清 PNG 后长按保存或发送，并将当前页面临时切换为挑战链接供右上角分享。

## 本地运行

```bash
npm install
npm run dev
```

默认地址：`http://127.0.0.1:5173/play-pack/`

## 构建

```bash
npm run build
```

生产基础路径为 `/play-pack/`。仓库根目录的 `play-pack/` 是 GitHub Pages 部署产物，不要手工编辑压缩文件。
