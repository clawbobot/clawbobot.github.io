# 这也能装？

移动优先的空间拼图游戏。玩家从传送带选择多格物品，旋转后放入箱子，在倒计时结束前完成 100% 装箱。

## 玩法

1. 点击传送带上的物品。
2. 点击箱格选择放置起点。
3. 使用左右旋转调整方向。
4. 确认青色预览有效后放入箱子。
5. 完成五轮逐步变难的可解订单。

每轮拼图都由完整布局反向生成，因此物品组合一定存在 100% 解。关卡会逐渐增加箱体尺寸、物品数量、形状复杂度，并缩短时间。

## 分享挑战 MVP

- 每关完成后生成个人战绩卡。
- 支持本地保存用户昵称、随机短标记和可选宣言。
- 战绩图片为 `1080 × 1440`，包含最终布局、成绩、二维码和挑战文案。
- 挑战链接携带关卡和分享者成绩，打开后可直接挑战同一订单。
- 挑战完成后比较双方用时和得分。
- 第一版无需账号或后端，挑战数据编码在链接中。

## 本地运行

```bash
npm install
npm run dev
```

默认地址：`http://127.0.0.1:5173/play-pack/`

## 构建

```bash
npm run build
```

生产基础路径为 `/play-pack/`。仓库根目录的 `play-pack/` 是 GitHub Pages 部署产物，不要手工编辑压缩文件。
